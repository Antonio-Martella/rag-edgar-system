# Creazione del file `.env`
Prima di tutto crea un file `.env` e scrivi:
```bash
SEC_USER_AGENT = "Nome Cognome email@email.com"
```

# Configurazione Github da teminale
```bash
git config --global user.email "tua_email@esempio.com"
```
```bash
git config --global user.name "tuo username github"
```

# Settaggio PATH Python
```bash
export PYTHONPATH=$PYTHONPATH:.
```

# Download pacchetti
```bash
pip install -r requirements.txt
```
